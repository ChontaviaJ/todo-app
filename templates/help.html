Help

This To-Do List application allows you to manage your tasks. You can add, delete tasks, view task analysis, generate reports, and more.

Features

  • Add a new task using the form on the main page.
  • Delete a task by clicking the "Delete" link next to the task.
  • View task analysis and word frequency in the analysis section.
  • Generate and download a PDF report of your tasks.
  • Logout from the app when you're done.
